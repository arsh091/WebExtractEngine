# Web Data Extractor

A modern web application to extract contact information (Phone, Email, Physical Address) from websites.

## Tech Stack
- **Frontend**: React.js (Vite), Tailwind CSS, GSAP
- **Backend**: Node.js, Express.js
- **Extraction**: Cheerio, Puppeteer
